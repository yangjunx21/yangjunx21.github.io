git clone 加速

这个可太重要了(参考了一篇博客，链接在本节最后)
全网我试过唯一有用的方法如下：gitclone.com！！！！！
gitclone.com是一个提供下载缓存的代码下载网站，使用方法十分简单，只需要在仓库地址前面加上 gitclone.com，就可以使速度提升几倍。

例如要克隆github上仓库地址https://github.com/killer-p/ctool.git
只需将地址改为https://gitclone.com/github.com/killer-p/ctool.git，在命令行中执行git clone https://gitclone.com/github.com/killer-p/ctool.git，速度直接起飞！芜湖！
官网首页：https://gitclone.com/

原文链接：https://blog.csdn.net/weixin_44821644/article/details/107574297

fairseq安装

  1. 按照github上面的流程走就好，但是有一个大坑，pip install --editable ./这个本地安装指令不是很好用的，需要连接到pypi，所以必须要指令一下url，要不然会因为非常慢卡在依赖建立那里，应该用：pip install --index-url https://pypi.tuna.tsinghua.edu.cn/simple --editable ./
