我就说小学期没选上python是最错误的人生结点啦

django 安装

python -m pip install Django

查看版本python -m django --version就好

python -m 和 python 的区别

-m会指定以脚本运行，这样的好处是sys.path会是当前的目录，在工程里面py文件被放到其他文件夹下面的时候，这样以脚本运行的方式自然会带来极大的方便

django-admin 和 manage.py

https://docs.djangoproject.com/zh-hans/4.1/ref/django-admin/
